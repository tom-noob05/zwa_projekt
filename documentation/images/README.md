# Screenshots for Product Documentation

This folder is intended to hold screenshots referenced in the product documentation.

Suggested filenames:
- home.png — hlavní stránka a výpis inzerátů
- register.png — registrační formulář
- create_offer.png — formulář vytvoření inzerátu
- offer_detail.png — detail inzerátu

Please add PNG/JPEG files here and update `product_documentation.md` if you want different names.
